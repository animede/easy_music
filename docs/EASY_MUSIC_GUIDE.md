## 🎯 このアプリでできること

Easy Music は **ACE-Step 1.5** というAI音楽生成エンジンを使って、ジャンルを選ぶだけで音楽を自動生成します。

- 🎵 **30ジャンル** — J-POP、Rock、Jazz、EDM、演歌、ボカロなど
- 🤖 **AI作詞** — テーマに合わせた歌詞を自動生成
- 😊 **ムード/ボーカルチップ** — 10種のムード〆8種のボーカルタイプをワンタップ指定、AIキャプション生成に自動反映
- 🎛️ **キャプション自動/手動** — ジャンル別に最適なキャプション（音楽指示文）を自動生成
- 🎼 **AI強化** — BPM・キー・拍子を自動推定
- 🔊 **インラインプレイヤー** — スペクトルビジュアライザー付き
- 🌏 **13言語対応** — 日本語、英語、中国語、韓国語など
- 🎶 **JUKEBOX自動演奏** — ジャンルを選ぶだけで次々と新しい曲を自動生成・連続再生
- 🪄 **おまかせ生成** — 自然言語で指示するだけで全パラメータをAIが自動決定
- 🚫 **ネガティブプロンプト** — 避けたい要素を指定して品質を向上
- ⚙️ **詳細設定** — Shift・Sampler（ODE/SDE）など上級者向けパラメータ
- 🔊 **音量調整** — 全プレーヤーで音量スライダー＋ミュート対応

## ⚡ 基本の使い方

### ステップ1: ジャンルを選ぶ

画面に並んだジャンルタイルから好きなものをクリックします。選択すると紫色にハイライトされます。

### ステップ2: 作詞設定を開く（任意）

画面の「**作詞設定**」アコーディオンをクリックして展開すると、以下の3つの設定が現れます。**すべて任意で、空欄でもOK**です。

#### 😊 詳細オプション（ムード / ボーカル）

ムードチップから曲の雰囲気、ボーカルチップから声のタイプを各1つずつ選べます。選んだ内容はAIキャプション生成に自動反映されます。

| ムード例 | ボーカル例 |
|---|---|
| 😊 明るい / 😢 切ない / 🔥 激しい / 🍃 穏やか | 👩 女声 / 👨 男声 / 🎤 力強い |
| ✨ 神秘的 / 💕 ロマンチック / 🎉 楽しい | 🌬️ ウィスパー / 🎵 ファルセット / 👥 ハーモニー |
| 🌙 メランコリック / 🏔️ 壮大 / 😎 クール | |

> 💡 キャプション自動（ON）のときはムード/ボーカルはACE-Step内部のヒントに加算されます。2回クリックすると選択解除できます。

#### 💬 テーマ / シナリオ

曲のイメージを書きます。**空欄でもOK** — AIが自動でテーマを考えてくれます。

**入力例:**
```
夏の海辺で恋人と過ごす思い出
```
```
雨の日にカフェで読書する午後
```

#### 🚫 ネガティブプロンプト

生成時に「避けたい要素」を指定できます。画像生成AIのネガティブプロンプトと同じ考え方です。

- **空欄の場合**: デフォルト値が自動適用 → `low quality, noisy, distorted, muddy, clipping, off-key, out of tune, amateur, poorly mixed`
- **効果**: ノイズ・音割れ・音程外れ・低品質ミキシングを抑制し、全体的な音質を向上
- **カスタマイズ**: 特定の楽器や音色を避けたい場合などに自由入力

> 💡 通常はデフォルトのままで十分です。特別な理由がない限り空欄にしておきましょう。

#### 🎵 歌詞（空欄ならAIが自動生成）

自分で歌詞を書くこともできます。**空欄ならAIが自動生成**します。**♻️ 再生成**ボタンで歌詞をクリアしてAIに再生成させることもできます。

### ステップ3: 「🎵 生成」ボタンを押す

「🎵 生成」ボタンをクリック。あとはAIにおまかせ！

**自動処理の流れ:**
1. テーマ生成（空欄の場合）
2. AI作詞（歌詞が空欄の場合）
3. キャプション生成（ジャンル情報を元に）
4. AI強化（BPM・キー・拍子を自動設定）
5. 音楽生成（1〜3分で完成）

### ステップ4: 再生

完成したらプレイヤーが表示されます。スペクトルビジュアライザーと一緒に楽しめます。

🔊 **音量調整**: プレーヤーのスピーカーアイコン横のスライダーで音量を調整できます。アイコンクリックでミュート/解除。音量設定はブラウザに保存され、次回アクセス時に復元されます。

## 🪄 おまかせ生成

「夏の海辺で聴きたい爽やかなポップ」のように**自然言語で入力するだけ**で、AIがジャンル・キャプション・歌詞・BPM・キーなど全パラメータを自動決定して音楽を生成します。

### 使い方

1. ジャンルタイルの上にある入力欄にテキストを入力
2. 「🎵 生成」ボタンを押す
3. AIが全てを自動決定して生成開始

### 🎲 ランダムボタン

入力欄横の **🎲** ボタンを押すと、ランダムなキャプション・歌詞・設定をフォームに自動入力します。「とりあえず何か聴きたい」ときに便利です。

> 💡 おまかせ生成はACE-Stepの `sample_query` APIを使用しています。ジャンルタイルを選ぶ通常の生成と比べ、LMが内部で全てを生成するため、より一貫性のある結果が得られます。

## 🎶 JUKEBOX 自動演奏モード

JUKEBOXは「ジャンルを選ぶだけでAIが次々と新しい曲を作り続ける」自動演奏機能です。ラジオのように、ボタン一つで無限に新しい音楽を楽しめます。

### 使い方

1. ヘッダーの **🎶 自動演奏** ボタンをクリックしてJUKEBOXモードに切替
2. 演奏したいジャンルタイルを複数選択（複数選択可）
3. **▶ JUKEBOX スタート** ボタンを押す
4. AIが選んだジャンルからランダムに曲を生成・再生
5. 1曲終わると自動で次の曲を生成・再生
6. **■ STOP** ボタンで停止

### JUKEBOXモードの特徴

- 🎲 **ランダム選曲**: 選択したジャンルから毎回ランダムに1つをピック
- 🤖 **完全自動**: テーマ・歌詞・BPM・キー等すべてAIが自動決定
- 🎬 **ビジュアライザー**: フルスクリーンオーバーレイで音楽を視覚的に楽しめる
- 🌟 **5種類のアニメーション**: Spectrum / Wave / Ring / Particles / Pulse から選択可能
- 🎨 **ジャンル背景**: 再生中のジャンルに合わせた背景画像を表示
- 📝 **歌詞表示**: 生成された歌詞をオーバーレイ上に表示
- 🔢 **再生カウント**: 何曲目かをリアルタイム表示

### ビジュアライザーのアニメーションモード

オーバーレイ左下のセレクターで切替えられます：

| モード | 説明 |
|--------|------|
| **Spectrum** | 周波数スペクトラム |
| **Wave** | 波形アニメーション |
| **Ring** | 円形ビジュアライザー |
| **Particles** | 音に反応するパーティクル |
| **Pulse** | 音量に合わせた脈動リング |

**JUKEBOX時のランダム切替:**
- ボタンを押さない場合 → 曲ごとに5種類からランダムに切替
- ボタンで指定した場合 → そのモードに固定（曲が変わっても維持）
- 次回JUKEBOX開始時 → 再びランダムにリセット

### 再生画面の操作

- **オーバーレイをクリック**: 操作パネルに戻ります（JUKEBOX中は音楽は止まらず再生続行）
- **🎵 再生画面に戻る** ボタン: 右下に表示されるフローティングボタンでビジュアライザーに復帰

### STOPボタンの動作

**■ STOP** を押すと、現在の曲を即座に停止してJUKEBOXを終了します。オーバーレイも自動的に閉じ、通常の操作画面に戻ります。

> 💡 **ヒント**: JUKEBOXはBGMとして流しっぱなしにするのに最適です。作業用BGM、パーティー、リラックスタイムなどに！

## ⚙️ 各設定の意味

### 長さ（秒）

| 値 | 用途 |
|-----|------|
| **Auto** | 歌詞の長さからLMが最適な長さを自動推定（**おすすめ**） |
| 30秒 | お試し・ショートクリップ |
| 60秒 | ショートバージョン |
| 120秒 | 標準的な長さ |
| 180秒 | フルサイズに近い |
| 240〜300秒 | 長めの曲 |

> 💡 v1.6からデフォルトがAutoになりました。歌詞の量に合わせて最適な長さが自動選択されます。

### 言語

歌詞の言語を選択します。日本語、英語、中国語、韓国語など13言語に対応。選んだ言語で歌詞が生成されます。

### 楽器のみ

チェックを入れるとボーカルなしのインストゥルメンタル楽曲を生成します。BGMや作業用音楽に最適。

> 💡 Trance・ムード・ドライビング・フォルクローレは自動的にインストゥルメンタルになります。

### キャプション自動

ジャンルに合った内蔵キャプション（音楽スタイルの指示文）を使用します。デフォルトでON。

- **ON**: ジャンルタイルに設定された最適なキャプションを使用（おすすめ）
- **OFF**: 外部LLMにキャプション生成を依頼（より創造的だが時間がかかる）。ムード/ボーカルチップの内容がキャプションに反映されます

### Thinking

キャプション生成スタイルの切替。

- **ON**（デフォルト）: ACE-Stepの内部思考が音楽を補完。短いヒント文を渡す。
- **OFF**: LLMが3√5文の詳細な制作指示書全文を生成。楽器・リズム・ボーカルスタイルまで詳細に指定したいときに使用

### STEP（品質ステップ数）

音楽生成の品質と速度のバランスを調整します。

| 値 | 用途 |
|-----|------|
| 8 | ターボ・高速テスト |
| 20 | 迅速・お試し |
| 50 | 標準（おすすめ） |
| 80 | 高品質 |
| 100 | 最高品質（時間がかかる） |

### 詳細情報ボタン

「詳細情報」ボタンを押すと、生成時に使用されたパラメータ（キャプション、歌詞、送信パラメータ）を確認できます。

## ⚙️ 詳細設定（上級者向け）

コントロールバーの「詳細設定」アコーディオンを開くと、以下の上級者向けパラメータにアクセスできます。

### Shift（タイムステップシフト）

フローマッチング（DiT）モデルのノイズスケジュールを制御します。値が低いと細部に、高いと全体構造に集中します。

| 値 | 効果 | 向いている用途 |
|---|---|---|
| **Auto** | モデルに応じたデフォルト値（**おすすめ**） | 通常はこれでOK |
| 1.0〜2.0 | ディテール段階に集中 → 細かい音質・テクスチャが精密 | 楽器の音色・ミキシングの繊細さ重視 |
| 3.0 | セマンティクス寄り → 構造がプロンプトに忠実 | Turbo推奨。ジャンル感・雰囲気重視 |
| 4.0〜5.0 | セマンティクス最大 → メロディ・リズムが強く制御 | 明確な構成を求める場合 |

> 💡 **Turboモデル**: デフォルト3.0。**Baseモデル**: 1.0〜2.0がおすすめ。

### Sampler（推論サンプラー）

音楽生成時のサンプリング方式を選択します。

| | ODE（推奨） | SDE |
|---|---|---|
| **仕組み** | 決定論的サンプリング | 確率的サンプリング（各ステップでノイズ再注入） |
| **再現性** | 同じseedで完全同一 | seedが同じでも微妙に異なる |
| **音の質感** | クリーン・整然 | 有機的・生演奏的な揺らぎ |
| **安定性** | 常に安定 | 少ステップ(8)ではノイズ増加のリスクあり |
| **推奨** | Turbo (8ステップ) | Base (50ステップ以上) |

> 💡 **初心者はODE（デフォルト）のままでOK**。SDE は Base モデル＋50ステップ以上で使うと有機的な質感が楽しめます。

### 音量調整

全プレーヤー（インライン・オーバーレイ・JUKEBOX）にスピーカーアイコンと音量スライダーが搭載されています。

- 🔊 **スピーカーアイコンクリック**: ミュート / ミュート解除
- **スライダー**: 音量を0〜100%で調整
- **設定保存**: 音量はlocalStorageに保存され、ブラウザを閉じても次回アクセス時に復元されます

## 🎨 もっと良い曲を作るコツ

### テーマの書き方

#### ❌ あいまいな例
```
楽しい曲
```

#### ✅ 具体的な例
```
春の桜並木を歩く若いカップル。希望に満ちた新生活のスタート。
```
```
真夜中の東京、ネオンに照らされた雨の路地を一人歩く。
```

**ポイント:**
- 場面や情景を具体的に書く
- 感情や雰囲気を添える
- 短くても大丈夫（1〜2文で十分）

### 歌詞を自分で書く場合

構造タグを使うと品質が上がります：

```
[verse]
朝日が昇る街の中
新しい一日が始まる

[chorus]
未来へ続く道を歩こう
どんな困難も乗り越えて

[verse]
雨の日も風の日も
心には灯がともる

[chorus]
未来へ続く道を歩こう
夢を叶える日まで
```

**使えるタグ:**
| タグ | 意味 |
|------|------|
| `[verse]` | Aメロ / バース |
| `[chorus]` | サビ / コーラス |
| `[pre-chorus]` | Bメロ |
| `[bridge]` | Cメロ / ブリッジ |
| `[intro]` | イントロ |
| `[outro]` | アウトロ |
| `[instrumental]` | 間奏 |

### ジャンル選びのヒント

- **初めて**: J-POP、Rock あたりが安定
- **BGM**: Lo-Fi、ボサノバ、ムード
- **インパクト重視**: EDM、Phonk、Trance
- **和風**: 演歌、民謡、70年代歌謡曲
- **おしゃれ**: シティポップ、Jazz、Acoustic Pop

## ❓ よくある質問

### Q: 生成にどのくらい時間がかかる？

60秒の曲で約1分、120秒の曲で約2分が目安です。サーバーの混雑状況により前後します。

### Q: テーマも歌詞も空欄で大丈夫？

はい、ジャンルさえ選べばAIが全て自動生成します。テーマがあるとより意図に合った曲になります。

### Q: 同じジャンルで何度も生成すると同じ曲になる？

いいえ、毎回ランダムに異なる曲が生成されます。

### Q: ボーカルなしの曲を作りたい

「楽器のみ」にチェックを入れてください。Trance・ムード・ドライビング・フォルクローレは自動的にインストになります。

### Q: 歌詞が英語になってしまう

「言語」設定を「日本語」に変更してください。

### Q: 外部LLMがないと使えない？

いいえ。外部LLM（Gemma等）に接続できない場合、自動的にローカルLLM（Qwen3-1.7B）にフォールバックします。初回はモデルのダウンロード（約1.1GB）に数分かかります。

### Q: エラーが出て生成できない

以下を確認してください：
1. ACE-Step APIサーバーが起動しているか
2. ネットワーク接続が安定しているか
3. ブラウザのコンソールにエラーが出ていないか
